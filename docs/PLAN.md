## MVP Goal App
- CRUD para metas y tareas.
- Solo poder crear tareas si están vinculadas a una meta.
- No poder repetir el nombre de una meta.
- API.
- Metas ordenadas por fecha de vencimiento, si no hay por fecha de creación.
- Tareas ordenadas por fecha de creación.
- Posibilidad de establecer una fecha de vencimiento.

## Tecnologías
### Backend
 Lenguaje: Typescript.
 Runtime: Node.
 Frameworks: Express.
 Base de datos: PostgreSQL.
 Dependencias: Prisma, Zod.

## Frontend
Lenguaje: Typescript.
Frameworks: Tailwind CSS, React
Dependencias: TanStackQuery
